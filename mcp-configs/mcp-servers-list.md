# MCP Servers Documentation

## 1. Rolldice MCP Server
Purpose: Demonstrates tool calling capability using random dice rolls.

Functionality:
- Generates random numbers
- Used to test AI tool interaction

---

## 2. Bootcamp AI Agent Server
Purpose: Bootcamp-specific MCP server used for training exercises.

Functionality:
- Provides workshop resources
- Enables AI agent workflow simulations

---

## 3. Calendar Booking MCP Server
Purpose: Allows AI to schedule meetings and check availability.

Functionality:
- Create calendar events
- Check available times
- Manage bookings

---

## 4. GitHub MCP Server
Purpose: Allows Claude to interact directly with GitHub repositories.

Functionality:
- Read repositories
- Create commits
- Manage issues
- Review code
